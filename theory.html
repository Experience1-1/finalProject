Глава 2. Циклы и условия. Операторы if, else, elif

Условные операторы if, else и elif используются в языке программирования Python для выполнения различных действий в зависимости от выполнения определенных условий. Они позволяют программисту создавать ветвления в программе, в которых один или несколько блоков кода выполняются только при определенных условиях. Оператор if проверяет истинность заданного условия и, если оно истинно, то выполняет блок кода, который следует за оператором if. Если условие ложно, то блок кода не будет выполнен.

Пример: age = 25 if age >= 18: print("You are an adult.") В этом примере, если значение переменной age равно или больше 18, то выведется сообщение "You are an adult." Оператор else используется в паре с оператором if. Если условие, указанное в операторе if, не выполняется, то выполняется блок кода, указанный в операторе else. Пример:

Иначе, если значение переменной age меньше 18, то выводится сообщение "You are not an adult yet." Оператор elif используется в цепочке if-else и позволяет проверять дополнительные условия в случае, если предыдущие не были выполнены.
